# ADB USB调试开启指南（通用）

## 通用步骤（所有Android手机）

### 第1步：开启开发者选项
1. 打开"设置"
2. 找到"关于手机"（通常在最底部）
3. 连续点击"版本号"**7次**
4. 出现"您现在处于开发者模式"提示

### 第2步：开启USB调试
1. 返回设置主页
2. 找到"开发者选项"（通常在"系统"或"其他设置"下）
3. 开启"USB调试"
4. 弹出确认框 → 点击"确定"

### 第3步：连接电脑
1. 用USB线连接手机和电脑
2. 手机上弹出"是否信任此电脑" → 点击"允许"
3. 电脑上运行 `adb devices` 验证

## 各品牌差异

### OPPO / Realme / OnePlus (ColorOS / OxygenOS)
- 关于手机 → 版本号（点7次）
- 设置 → **其他设置** → 开发者选项 → USB调试
- 注意：ColorOS可能需要输入验证码

### 小米 / Redmi (MIUI / HyperOS)
- 设置 → 我的设备 → 全部参数 → **MIUI版本**（点7次）
- 设置 → **更多设置** → 开发者选项 → USB调试
- 注意：需要登录小米账号才能开启

### 华为 / 荣耀 (HarmonyOS / EMUI)
- 设置 → 关于手机 → 版本号（点7次）
- 设置 → **系统和更新** → 开发者选项 → USB调试
- 注意：HarmonyOS 4.0+ 可能在"系统"下

### 三星 (One UI)
- 设置 → 关于手机 → 软件信息 → **编译编号**（点7次）
- 设置 → 开发者选项（直接在设置主页底部）→ USB调试

### vivo / iQOO (OriginOS / FuntouchOS)
- 设置 → 关于手机 → **软件版本号**（点7次）
- 设置 → **系统管理** → 开发者选项 → USB调试

### Google Pixel (原生Android)
- 设置 → 关于手机 → **版本号**（点7次）
- 设置 → 系统 → 开发者选项 → USB调试

## 常见问题

| 问题 | 解决 |
|------|------|
| 找不到"开发者选项" | 没有点够7次版本号，或需要重启设置APP |
| 电脑不识别手机 | 换USB线/换USB口/安装手机驱动 |
| 没有弹出信任提示 | 撤销USB调试授权后重新连接 |
| adb devices显示unauthorized | 手机上点击"信任此电脑" |
| 手机只充电不传数据 | 下拉通知栏 → USB连接模式改为"文件传输" |
