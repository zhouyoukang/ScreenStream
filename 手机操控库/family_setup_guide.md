# 📱 远程协助设置指南（给家人看的）

> 设置一次，以后随时可以远程帮你解决手机问题。全程5分钟。

---

## 第一步：安装两个APP（2分钟）

### 1. ScreenStream（远程操控核心）
- 打开手机**应用商店**
- 搜索 **"ScreenStream"**
- 安装后打开，按提示允许所有权限
- **重要**：弹出的"无障碍服务"必须开启（这是远程操控的核心）

### 2. Tailscale（跨网络连接）
- 打开手机**应用商店**  
- 搜索 **"Tailscale"**
- 安装后打开，用我发给你的**登录链接**登录
- 登录后保持后台运行即可

---

## 第二步：简单设置（2分钟）

### ScreenStream设置
1. 打开 ScreenStream
2. 点击右上角 **开始**（绿色按钮）
3. 允许"显示在其他应用上方"
4. 允许"屏幕录制"
5. 看到显示 **IP地址和端口号** 就表示成功了

### 让APP自动启动
1. 打开手机**设置** → **电池**
2. 找到 **ScreenStream** 和 **Tailscale**
3. 关闭"电池优化"（让它们在后台持续运行）

> 不同手机操作略有不同：
> - **华为/荣耀**：设置 → 电池 → 启动管理 → 找到APP → 关闭自动管理，手动打开所有开关
> - **小米/红米**：设置 → 电池 → 省电优化 → 找到APP → 无限制
> - **OPPO/一加**：设置 → 电池 → 更多电池设置 → 优化电池使用 → 找到APP → 不优化
> - **vivo**：设置 → 电池 → 后台高耗电 → 找到APP → 允许

## 常见问题

**Q: 安全吗？别人会不会看到我手机？**
A: 安全的。只有通过Tailscale登录的设备（就是我的电脑）才能连接。其他人无法访问。

**Q: 会不会很费电？**
A: ScreenStream在后台几乎不耗电。只有我远程连接时才会稍微增加耗电。

**Q: 我不想被远程控制了怎么办？**
A: 关掉ScreenStream APP就行，或者直接在通知栏点击"停止"。

**Q: 手机重启后还能用吗？**
A: Tailscale会自动重连。ScreenStream需要你手动打开一次（点击开始按钮）。
