# Agent 视频工坊 — 升维指令

---

## 道 · 顶层目的

**把散落在三块硬盘里的原始素材，变成可以直接发布的成品视频，全程不需要人打开剪辑软件。**

AI 做所有苦力活（扫描、评分、截取、拼接、配音、字幕、BGM），人只做一个决定：发还是不发。

---

## 法 · 必达标准

任何产出的视频必须满足：

- **可直接发布** — 有封面、有字幕、有BGM、有合理的时长（2-8分钟），拿到手就能上传平台
- **声画完整** — 不允许出现连续3秒以上的纯静音画面（除非是故意的留白）；不允许出现黑帧/花屏/卡顿
- **质量可量化** — 每个视频有五维评分（画质/音频/节奏/时长/内容），分数和问题写入 `quality_report.json`
- **源可追溯** — 每个成品都能追溯到源素材路径和处理参数，不丢失来源信息
- **一键复现** — `python pipeline.py build<N>` 或 `python smart_cut.py all` 从零重建成品，中间产物可清理可重生
- **前端可预览** — `python pipeline.py serve` 启动后，所有成品在浏览器里可播放、可对比、可下载

### 各项目专项标准

| 项目 | 底线 |
|------|------|
| P1 B站科技视频 | TTS语音+字幕+BGM三件齐全，4K截图清晰度 |
| P2 录屏智剪 | 剔除所有等待/空闲/报错片段，只留操作精华 |
| P3 相机AI剪辑 | 94%静音已修复（必须有BGM），Ken Burns运镜自然 |
| P4 智能精华 | 50素材评分→Top15提取→无缝拼接，节奏流畅 |
| P5 旅行电影 | 电影感调色+运镜+BGM，叙事完整有开头有结尾 |

## 术 · 执行边界

### 可以做

- 读取和修改本目录内所有 `.py` / `.html` / `.md` / `.json` 文件
- 调用 `ffmpeg` / `ffprobe` 处理视频音频
- 调用 `edge-tts` 生成语音
- 启动 HTTP 服务器预览
- 扫描源目录（只读，不修改原始素材）

### 不能做

- **不修改源素材**（`E:\VideoEdit\` / `D:\屏幕录制\` / `E:\道\道生一\_video\` 只读）
- **不引入重型依赖**（不装 moviepy/opencv/tensorflow，只用 ffmpeg + 标准库）
- **不硬编码发布平台凭据**（B站账号/密码不进代码）
- **不生成超过 500MB 的单个文件**（压缩或分段）
- **不在 pipeline.py 中留死代码**（每个函数必须被某个命令调用）

### 工具链

| 工具 | 用途 | 约束 |
|------|------|------|
| `ffmpeg` | 视频处理核心 | 必须在 PATH 中 |
| `ffprobe` | 元数据提取 | timeout=10s |
| `edge-tts` | TTS 语音合成 | 中文用 zh-CN-XiaoxiaoNeural |
| `http.server` | 预览服务 | 端口 9876 |
| `shutil` | 文件汇聚/清理 | 只复制不移动源文件 |

### 目录约定

```
AI视频剪辑/
├── pipeline.py           ← 统一入口（status/build/collect/serve/analyze/clean）
├── smart_cut.py          ← 智能精华提取器（scan/cut/assemble/all）
├── travel_film.py        ← 旅行电影生成器
├── index.html            ← 前端展示（Tab式，暗色主题）
├── quality_report.json   ← 五维质量报告（自动生成）
├── 01-B站科技视频/      ← P1 成品 + 配套素材
├── 02-录屏智剪/          ← P2 成品
├── 03-相机AI剪辑/        ← P3 成品
├── 04-智能精华/          ← P4 成品 + 评分报告 + 片段
└── 05-旅行电影/          ← P5 成品 + 中间产物
```

### FFmpeg 踩坑备忘

- `zoompan` 在 Windows 上只用 `min(zoom+delta,target)`，不用 `if(eq(on,1),...)` 复杂表达式
- `subtitles` filter 中文路径会失败 → 先复制 SRT 到 TEMP 目录用 ASCII 路径
- BGM 可用 `sine` + `anoisesrc` lavfi 合成，不依赖外部素材
