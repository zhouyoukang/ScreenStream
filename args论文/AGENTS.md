# AGENTS.md — ARGs论文

## 目录用途
艾比湖荒漠土壤抗生素抗性基因(ARGs)SRT研究论文。
数据分析 + 可视化 + 论文撰写。

## 技术栈
- **语言**: Python 3.10+
- **数据分析**: pandas, numpy, scipy, scikit-learn
- **可视化**: matplotlib, seaborn
- **统计**: PERMANOVA, 相关性分析

## 关键文件
| 文件 | 用途 | 修改风险 |
|------|------|---------|
| `config.py` | 数据路径配置 | 🟢低 |
| `论文/SRT_论文完整框架.md` | 论文框架 | 🟡中 |
| `论文/SRT_论文完整版.docx` | 论文正文 | 🟡中 |
| `figures/` | 统计结果+图表 | 🟢低 |
| `数据/ph_swc_ec.xlsx` | 原始数据 | 🔴高 — 不可修改 |
