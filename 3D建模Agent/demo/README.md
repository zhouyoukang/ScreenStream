## 组件结构

### 杯身 (`body`)
- **功能**: 主容器，盛放液体
- **几何**: 锥形圆柱(difference挖空) + 外翻唇
- **特征**: 底部微收(taper=0.9)、顶部外翻唇、壁厚3mm可打印

### 把手 (`handle`)
- **功能**: 握持手柄
- **几何**: 4段hull()连接的D形弧线
- **特征**: 上下连接杯身、中部外凸28mm、截面12×8mm

## 使用方法

### 命令行修改参数
```bash
D:\openscad\openscad.com -o tall_mug.stl -D "body_height=130" -D "body_diameter=70" coffee_mug.scad
```

### OpenSCAD GUI
1. 打开 `coffee_mug.scad`
2. 右侧 Customizer 面板调整参数
3. F5预览 → F6渲染 → File > Export STL
