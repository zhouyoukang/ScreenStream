## 组件结构

### 头部 (`head`)
- **几何**: 扁球体 (scale椭球) + 面罩凹槽/面板 + 圆柱眼 + 腮红
- **CSG**: difference(球, 面罩槽) + union(面板, 眼, 腮红)

### 天线 (`antenna`)
- **几何**: 细圆柱杆 + 球体顶

### 躯干 (`torso`)
- **几何**: hull(上层宽圆柱, 下层窄圆柱) = 收腰效果 + 胸徽(同心圆柱)

### 手臂 (`arm`)
- **几何**: 肩球 + 上臂圆柱 + 肘球 + 方块手 + 手指槽
- **参数化**: `arm_angle` 控制姿态

### 腿部 (`leg`)
- **几何**: 髋球 + 大腿圆柱 + 膝球 + 小腿圆柱 + 方块脚掌

## 使用方法

### 命令行修改参数
```bash
D:\openscad\openscad.com -o big_robot.stl -D "scale_factor=2" -D "arm_angle=45" chibi_robot.scad
```

### OpenSCAD GUI
1. 打开 `chibi_robot.scad`
2. Customizer面板 → 拖动参数
3. F5预览 → F6渲染 → Export STL

### 打印建议
- 层高: 0.12-0.2mm
- 填充: 15-20%
- 支撑: 需要（手臂下方）
- 材料: PLA/树脂均可
- 推荐尺寸: scale_factor=1.5 (150%放大, ~150mm高)
