# 代码审查 Skill

> **☶艮·知止无为** — 知道何时停止、何时不动、何时等待

## 概述

- **用途**: 多维度代码审查，发现问题并给出改进建议
- **触发**: 用户要求审查代码时
- **卦象**: ☶艮（知止无为）

## 审查维度

| 维度 | 检查点 |
|------|--------|
| 正确性 | 功能正确、逻辑无误 |
| 健壮性 | 边界条件、空值处理 |
| 性能 | 复杂度、内存、计算效率 |
| 安全 | 注入风险、凭据安全 |
| 可读性 | 命名清晰、注释恰当 |
| 可维护性 | 耦合度、内聚度 |

## 审查输出格式

```markdown
## 发现问题
### 严重 [🔴]
- 问题1: 描述 + 位置
### 建议 [🟡]
- 建议1: 描述 + 位置
### 优点 [🟢]
- 优点1
## 总结
[总体评价]
```

## 注意事项

- 只指出问题，不强制修改
- 提供修复建议，不代替用户做决定
- 优点也要肯定，保持平衡
- 审查代码前先完整读取
