💡 前置要求和常见问题

前置要求：

  1. 电脑安装 Python 3.8+（python.org，勾选Add to PATH）
  2. 电脑安装 ADB平台工具（developer.android.com，解压到D:\platform-tools）

常见问题：

  • “连接码未找到” → 确认电脑端已运行并显示相同连接码
  • “0台设备” → 检查USB线连接，手机弹窗授权调试
  • “无障碍未开启” → 需在手机设置手动开启（无法通过ADB自动开启）

3
权限配置

ScreenStream需要以下权限才能正常工作。请在手机上逐项开启：

📺 屏幕录制权限 必需
  1. 打开ScreenStream → 点击"开始"
  2. 系统弹窗"开始录制屏幕" → 点击"立即开始"
  3. 首次会请求通知权限，建议允许
♿ 无障碍服务 反向控制必需
  1. 手机设置 → 无障碍（或搜索"无障碍"）
  2. 找到 ScreenStream → 开启
  3. 确认弹窗中选择"允许"
🔄 Android 13+ 受限设置（点击展开）
🔄 解除受限设置 Android 13+ 必需
  1. 手机设置 → 应用 → 找到 ScreenStream
  2. 点击右上角 ⋮ 菜单 → "允许受限设置"
  3. 验证PIN/指纹后，回到无障碍页面重新开启

Android 13及以上版本，通过APK安装的应用默认被限制使用无障碍服务。必须先解除限制才能开启。

🔋 电池优化 推荐
  1. 手机设置 → 电池 → 应用省电管理
  2. 找到 ScreenStream → 选择"不限制"
  3. 防止系统杀后台导致投屏中断
