        "display_width_px": 1920,

## 🎓 第二梯队：学术研究项目（开源）

### 1. Microsoft UFO³ ⭐⭐⭐⭐⭐

**GitHub**：https://github.com/microsoft/UFO/
**论文**：https://arxiv.org/abs/2402.07939

**核心能力**：

- UI-Focused Windows OS Agent
- 双Agent架构（AppAgent + ActAgent）
- 基于GPT-4V的视觉理解
- 支持跨应用任务流

**架构**：

```
用户请求
    ↓
HostAgent（任务规划）
    ↓
AppAgent（选择应用）
    ↓
ActAgent（执行操作）
    ↓
完成任务
```

**性能**：

- Windows原生应用：**90%+**准确度
- 跨应用任务：**85%+**成功率
- 平均响应时间：**3-5秒/步**

**使用示例**：

```python
from ufo import UFOAgent

agent = UFOAgent(
    model="gpt-4-vision-preview",
    api_key="your-key"
)

agent.execute("在记事本中写一封邮件，然后用Outlook发送")
```

**优势**：

- ✅ **开源免费**
- ✅ **微软研究**：技术先进
- ✅ **跨应用**：可以操作多个软件
- ✅ **详细文档**：易于上手

**劣势**：

- ⚠️ **依赖GPT-4V**：需要OpenAI API
- ⚠️ **速度**：较慢（每步3-5秒）

**适用场景**：研究、学习、个人项目

### 2. Microsoft OmniParser ⭐⭐⭐⭐⭐

**GitHub**：https://github.com/microsoft/OmniParser
**网站**：https://microsoft.github.io/OmniParser/

**核心能力**：

- **纯视觉**的GUI解析工具
- 将UI截图转为结构化元素
- 识别可交互图标
- 提供坐标和描述

**工作流程**：

```
截图
    ↓
OmniParser解析
    ↓
输出：
  - 元素列表（按钮、输入框等）
  - 每个元素的坐标
  - 元素的文字描述
    ↓
LLM决策
    ↓
执行操作
```

**性能**：

- ScreenSpot Pro：**39.5%**（SOTA）
- 小图标检测：**V1.5改进**
- 支持：Windows/Web/Mobile

**使用示例**：

```python
from omniparser import OmniParser

parser = OmniParser()
elements = parser.parse(screenshot)

for elem in elements:
    print(f"{elem.type}: {elem.text} at ({elem.x}, {elem.y})")
```

**优势**：

- ✅ **纯视觉**：不依赖UI Automation
- ✅ **通用**：支持所有平台
- ✅ **开源**：可以本地部署
- ✅ **HuggingFace模型**：易于集成

**劣势**：

- ⚠️ **需要LLM配合**：单独使用只是解析工具
- ⚠️ **准确度有限**：复杂界面可能漏检

**适用场景**：构建自己的GUI Agent

## 💻 第三梯队：开源工具库

### 1. OpenAdapt.AI ⭐⭐⭐⭐⭐

**GitHub**：https://github.com/OpenAdaptAI/OpenAdapt
**网站**：https://openadapt.ai/

**核心能力**：

- **录制-回放**自动化
- AI理解操作意图
- 自适应UI变化
- **完全免费**

**工作流程**：

```bash
# 1. 录制操作
openadapt record
# 执行你的操作...

# 2. 回放
openadapt replay
# AI自动重现操作
```

**特点**：

- 本地运行，保护隐私
- 跨平台支持
- 模型无关
- 生成式自动化

**优势**：

- ✅ **完全免费**：无API费用
- ✅ **简单易用**：录制即可
- ✅ **本地运行**：隐私安全
- ✅ **开源**：可定制

**劣势**：

- ⚠️ **需要预录制**：每个任务都要录一次
- ⚠️ **准确度**：~80%（低于商业方案）

**适用场景**：重复性任务、个人使用、预算有限

### 2. Pywinauto ⭐⭐⭐⭐

**GitHub**：https://github.com/pywinauto/pywinauto

**核心能力**：

- Windows UI Automation封装
- 元素级控制
- 支持Win32和UIA两种后端
- Python库

**使用示例**：

```python
from pywinauto import Application

# 启动应用
app = Application().start("notepad.exe")

# 查找窗口
window = app.window(title="无标题 - 记事本")

# 操作元素
window.Edit.type_keys("测试文字", with_spaces=True)
```

**优势**：

- ✅ **稳定**：元素级操作
- ✅ **Python生态**：易于集成
- ✅ **免费**：BSD许可证
- ✅ **成熟**：久经考验

**劣势**：

- ⚠️ **仅Windows**：不跨平台
- ⚠️ **学习曲线**：需要理解UI结构
- ⚠️ **网页应用**：支持有限

**适用场景**：Windows原生应用自动化

### 3. PyAutoGUI ⭐⭐⭐

**文档**：https://pyautogui.readthedocs.io/

**核心能力**：

- 跨平台GUI自动化
- 鼠标键盘模拟
- 图像识别（可选）
- 最简单的方案

**使用示例**：

```python
import pyautogui

# 点击
pyautogui.click(100, 200)

# 输入
pyautogui.write('Hello World')

# 截图
screenshot = pyautogui.screenshot()

# 图像识别
pos = pyautogui.locateOnScreen('button.png')
```

**优势**：

- ✅ **极简单**：5分钟上手
- ✅ **跨平台**：Win/Mac/Linux
- ✅ **免费**：BSD许可证
- ✅ **轻量**：无依赖

**劣势**：

- ⚠️ **坐标依赖**：分辨率变化就失效
- ⚠️ **不稳定**：UI变化就出错
- ⚠️ **无语义**：不理解界面含义

**适用场景**：简单脚本、快速原型
