## 三、VaM ↔ Voxta 通信协议

### 插件架构 (AcidBubbles.Voxta.83.var)

```
Voxta.cs (63.6KB) ─── 主入口, VaM Scripter插件
├── VoxtaClient.cs (34.7KB) ─── 核心通信客户端
│   ├── SignalRClient.cs (3.9KB) ─── SignalR协议(主)
│   └── WebSocketClient.cs (17.3KB) ─── WebSocket协议(备)
├── ActionsParser.cs (6.2KB) ─── 动作解析(表情/动画)
├── SpeechPlayback.cs (5.9KB) ─── TTS音频播放
├── VoxtaCredentials.cs (4KB) ─── 连接凭据
├── UI.cs (5KB) ─── VaM内UI面板
├── VaM/
│   ├── TriggerInvoker.cs (2.9KB) ─── 触发器调用
│   ├── VamAssets.cs (1.8KB) ─── 资源管理
│   └── SimpleTrigger.cs (2.6KB) ─── 简单触发器
└── Components/
    ├── AudioAtomTarget.cs (3.7KB) ─── 音频挂载到Atom
    └── SubtitlesAtomTarget.cs (5.9KB) ─── 字幕显示
```

### 通信流程

```
用户说话 → NAudio采集 → Vosk/Whisper STT → 文本
  → BuiltInTextReplacements (纠正)
  → BuiltInReplyPrefixing (防重复)
  → DashScope qwen-plus LLM (生成回复)
  → ActionsParser (解析动作标签)
  → EdgeTTS/F5TTS (语音合成)
  → SpeechPlayback → AudioAtomTarget (VaM内播放)
  → SubtitlesAtomTarget (显示字幕)
```
