## 打印参数建议

| 参数 | 推荐值 |
|------|--------|
| **材料** | PETG (耐热) 或 PLA+ |
| **层高** | 0.2mm |
| **填充** | 30-50% |
| **壁厚** | 3层+ |
| **支撑** | 视部件而定 |
| **打印温度** | PETG: 230-240°C / PLA: 200-210°C |
| **热床温度** | PETG: 80°C / PLA: 60°C |

## 打印注意

1. **受力部件** (臂、连杆) 建议50%填充、PETG材料
2. **外壳部件** 可用30%填充、PLA
3. 打印方向影响强度 — 力的方向垂直于层纹最弱
4. 建议先打印测试件验证尺寸精度
