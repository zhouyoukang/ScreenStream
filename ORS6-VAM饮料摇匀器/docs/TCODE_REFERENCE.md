# TCode协议参考

> 来源: [multiaxis/TCode-Specification](https://github.com/multiaxis/TCode-Specification)

## 概述

TCode (Toy Code) 是用于控制成人玩具的UART串口通信协议，
受G-code (3D打印机控制协议) 启发设计。

## 命令格式

```
<轴类型><通道><位置>[I<间隔ms>][S<速度>]
```

### 轴类型
| 前缀 | 类型 | 说明 |
|------|------|------|
| L | Linear | 线性运动 |
| R | Rotation | 旋转运动 |
| V | Vibration | 振动 |
| A | Auxiliary | 辅助功能 |

### 通道
0-9, 每种轴类型最多10个通道

### 位置
- **4位格式**: 0000-9999 (最常用)
- **3位格式**: 000-999
- **2位格式**: 00-99
- 5000 = 中位/默认

### 运动修饰符
- `I<ms>` — 间隔时间: 到达目标位置的时间(毫秒)
- `S<speed>` — 速度: 每秒移动的位置单位

## SR6 轴定义

| 轴 | 代码 | 运动 | 说明 |
|----|------|------|------|
| Stroke | L0 | 上下 | 主行程轴 |
| Surge | L1 | 前后 | 推进 |
| Sway | L2 | 左右 | 摆动 |
| Twist | R0 | 旋转 | 绕主轴旋转 |
| Roll | R1 | 横滚 | 左右倾斜 |
| Pitch | R2 | 俯仰 | 前后倾斜 |

## 命令示例

```
L09999I1000    # L0轴移到最大位置，用时1000ms
L00000I500     # L0轴移到最小位置，用时500ms
L05000         # L0轴立即移到中位
R09999I2000    # R0轴旋转到最大，用时2000ms

# 多轴同时 (空格分隔)
L09999I500 R05000I500 L15000I500
```

## 设备命令

| 命令 | 功能 |
|------|------|
| D0 | 停止所有运动 |
| D1 | 全轴归中位 |
| D2 | 查询设备信息 |

## 通信参数

| 参数 | 值 |
|------|-----|
| 波特率 | 115200 |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验 | 无 |
| 行结尾 | \n |

## 通信方式

### USB串口
最可靠，延迟最低 (~1ms)

### WiFi UDP
ESP32固件内建Web配置页，默认UDP端口8000

### Bluetooth LE
使用Nordic UART Service (NUS):
- Service: `6e400001-b5a3-f393-e0a9-e50e24dcca9e`
- TX (写): `6e400002-b5a3-f393-e0a9-e50e24dcca9e`
- RX (通知): `6e400003-b5a3-f393-e0a9-e50e24dcca9e`
