## 修改指引

- **改控制逻辑** → 去 `远程桌面/` 修改 remote_agent.py
- **改前端交互** → 去 `远程桌面/` 修改 remote_desktop.html
- **改文件共享** → 去 `构建部署/` 修改 file_share.py
- **改穿透配置** → 去 `远程桌面/frp/` 修改 frpc.toml
- **更新连接信息** → 修改本目录 `连接卡.md`
- **更新架构记录** → 修改本目录 `README.md`
