## 🚀 MiGPT 智能AI集成

### 快速开始

MiGPT是一个强大的Home Assistant集成，让您的小爱音箱拥有AI对话能力！

**查看使用手册**：
```
📚_MiGPT集成文档/🎯MiGPT使用手册_最终版.md
```

### 主要功能

- ✅ **AI智能对话** - 与小爱音箱进行自然对话
- ✅ **语音播报** - 自动化语音通知
- ✅ **智能自动化** - 早安问候、天气播报、提醒等
- ✅ **多AI支持** - 支持OpenAI、DeepSeek、智谱AI等

### 快速测试

在 **开发者工具 → 服务** 中执行：

```yaml
service: xiaomi_mibot.chat
data:
  message: "给我讲个笑话"
  send_to_device: true
```

## 🆘 帮助与支持

### 常见问题

1. **MiGPT登录失败**  
   → 查看 `⚠️登录问题解决方案.md`

2. **AI对话没反应**

```

```

### 重启系统

```bash
# Docker方式
docker restart homeassistant

# 或在界面中
设置 → 系统 → 重启
```
