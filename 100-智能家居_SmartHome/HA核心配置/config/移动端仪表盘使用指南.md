# 📱 移动端智能家居仪表盘使用指南

## 🎯 概述

这是一个专为移动设备优化的Home Assistant仪表盘，提供了直观、高效的智能家居控制体验。界面采用现代化设计，支持触摸友好的交互方式。

## 📋 功能特性

### 🏠 首页 (Home)
- **顶部状态栏**: 实时显示天气、时间、功率消耗和电池状态
- **快捷场景**: 一键切换回家/离家模式
- **环境监控**: 温度、湿度、光照度实时监控
- **照明控制**: 主照明和飞利浦氛围灯带控制
- **电源监控**: 实时功率图表和移动电源状态
- **智能设备**: 电饭煲、风扇等设备状态

### 🤖 AI助手 (AI Assistant)
- **服务状态**: OpenAI助手和MiGPT服务监控
- **智能对话**: 支持语音识别的AI聊天界面
- **音箱控制**: 小爱音箱媒体播放控制

### 📹 监控 (Monitoring)
- **摄像头画面**: 小米智能摄像机实时监控
- **存储状态**: 存储空间和SD卡状态监控

### 🖨️ 3D打印 (3D Printing)
- **打印状态**: 拓竹A1打印机实时状态
- **打印进度**: 当前层数和总层数显示
- **打印腔体**: 实时摄像头画面

### ⚙️ 设置 (Settings)
- **快捷设置**: 灯光模式和主要开关控制
- **电池监控**: 所有设备电池状态汇总

## 🎮 场景模式

### 🏠 回家模式
- 自动开启主要照明
- 设置氛围灯为暖白光
- 启动风扇适中档位
- 开启必要插座设备

### 🚪 离家模式
- 关闭所有照明设备
- 关闭风扇和插座
- 确保安全离家

### 🌙 睡眠模式
- 关闭主照明
- 氛围灯调整为夜灯模式（暖橙色，低亮度）
- 风扇调至静音档位

### 💼 工作模式
- 开启明亮照明
- 设置冷白光提高专注度
- 风扇适中档位保持舒适

### 🎬 娱乐模式
- 关闭主照明
- 氛围灯设为彩色模式
- 风扇静音运行

### 🧹 清洁模式
- 开启最亮照明
- 风扇最大档位通风
- 适合打扫清洁时使用

### 🌱 节能模式
- 最小化能耗设置
- 仅保留必要照明
- 关闭非必需设备

### 🚨 紧急模式
- 开启所有照明
- 红色闪烁警示
- 紧急情况快速响应

## 📱 移动端优化特性

### 🎨 视觉设计
- **圆角卡片**: 16px圆角，现代化外观
- **渐变背景**: 根据设备类型使用不同渐变色
- **动态颜色**: 根据状态自动调整图标颜色
- **响应式布局**: 适配不同屏幕尺寸

### 👆 交互体验
- **触摸友好**: 大按钮设计，易于点击
- **手势支持**: 支持滑动和长按操作
- **即时反馈**: 点击时有视觉反馈
- **加载动画**: 平滑的状态转换

### 🎯 性能优化
- **懒加载**: 按需加载内容
- **缓存策略**: 优化资源加载
- **离线支持**: 基础功能离线可用

## 🚀 使用方法

1. **访问仪表盘**: 在Home Assistant侧边栏选择"移动端优化版"
2. **场景切换**: 点击首页的场景按钮快速切换模式
3. **设备控制**: 点击设备卡片进行开关控制
4. **详细信息**: 长按卡片查看详细信息
5. **AI对话**: 在AI助手页面进行语音或文字交互

## 🔍 故障排除

### 常见问题

1. **卡片显示异常**
   - 检查实体名称是否正确
   - 确认设备在线状态
   - 重启Home Assistant服务

2. **场景无法执行**
   - 检查scripts.yaml配置
   - 确认目标实体可用
   - 查看Home Assistant日志

3. **AI助手无响应**
   - 检查AI服务状态
   - 确认网络连接正常
   - 重启相关服务

### 性能优化建议

1. **定期清理日志**: 保持系统运行流畅
2. **更新组件**: 使用最新版本的自定义卡片
3. **监控资源**: 关注CPU和内存使用情况
