## 一、大模块（🔴 需用户决策）

### S1. 远程家庭协助 ⭐⭐⭐
> **场景**：帮远方父母/老人配置手机、解决问题、安装APP、调整设置
> **用户痛点**：老人不会操作、电话沟通效率低、需要"看到并操控"对方屏幕
> **参考**：TeamViewer Host(无人值守)、小米远程协助、AirMirror

| 子功能 | 难度 | 说明 |
|--------|------|------|
| 跨网P2P连接 | 🔴高 | STUN/TURN/ICE + WebRTC信令，需要中继服务器 |
| 无人值守模式 | 🟡中 | 被控端自启动、自连接、免确认 |
| 语音通话覆层 | 🟡中 | WebRTC音频通道，边操作边语音指导 |
| 引导指针模式 | 🟢低 | 在对方屏幕画圈/箭头指引操作位置 |
| 一键求助按钮 | 🟢低 | 老人按一个按钮即发起远程协助请求 |
| 操作录像回放 | 🟡中 | 录制整个协助过程，方便老人重复学习 |

**开发估算**：核心P2P 2-3周 + 辅助功能 1周
**差异化**：市面产品要么收费(TeamViewer)要么限制品牌(小米)，开源免费方案有巨大需求

---

### S2. 手机防盗追踪 ⭐⭐⭐
> **场景**：手机丢失/被盗后，远程定位、拍照取证、锁定、数据擦除
> **用户痛点**：Google Find My Device功能有限，Cerberus收费
> **参考**：Cerberus、CrookCatcher、Prey、Google Find My Device

| 子功能 | 难度 | 说明 |
|--------|------|------|
| GPS定位上报 | 🟡中 | 定时上报GPS到服务器/邮件 |
| 远程拍照取证 | 🟡中 | 前后摄像头远程拍照+上传 |
| 远程录音取证 | 🟡中 | 麦克风远程录音+上传 |
| 解锁失败自拍 | 🟢低 | 密码错误3次自动拍照+发邮件 |
| SIM卡更换报警 | 🟡中 | 检测SIM变化立即发送新号码+位置 |
| 远程锁定/擦除 | 🔴高 | Device Admin权限，远程触发 |
| 地理围栏 | 🟡中 | 设置安全区域，离开时报警 |
| 假关机 | 🟡中 | 模拟关机界面但保持后台运行 |
| 防卸载保护 | 🔴高 | Device Admin + 系统应用级保护 |

**开发估算**：核心追踪 2周 + 高级功能 2周
**差异化**：开源防盗App极少，Cerberus收费且停止更新

---

### S3. VR投屏与控制 ⭐⭐
> **场景**：手机投屏到VR头显，在VR中用虚拟大屏玩手游/看视频
> **参考**：Spatial Phone、Virtual Desktop、ALVR

| 子功能 | 难度 | 说明 |
|--------|------|------|
| WebXR投屏界面 | 🔴高 | VR浏览器中渲染3D空间+浮动屏幕 |
| VR手柄映射触控 | 🔴高 | Quest手柄射线→手机触控坐标 |
| 可调虚拟屏幕 | 🟡中 | 大小/位置/曲率/距离可调 |
| 低延迟编码 | 🟡中 | H265硬编码+VR优化 |
| 多屏模式 | 🟡中 | VR中同时显示多个窗口 |

**开发估算**：WebXR基础 3-4周
**差异化**：Spatial Phone收费$8/月，开源VR投屏方案几乎没有
