# 凭据中心 — 多Agent共享核心数据的唯一真相源

> **一句话**：所有Agent/脚本/配置读取凭据前，先读此文件获取结构，再从 `secrets.env` 获取实际值。
>
> **辩证统一**：公开的是结构（此文件），隐藏的是值（secrets.env）。知道有什么，与知道是什么，分离。

## 二、解决方案：凭据中心架构

### 设计原则

```
公开的是结构 → 凭据中心.md（git tracked，所有Agent可read_file）
隐藏的是值   → secrets.env（gitignored，Agent用run_command读取）
禁止的是散落 → Memory/文档/.md禁止存储实际凭据值
```

### 四层架构

```
┌─────────────────────────────────────────────────────────┐
│  Layer 4: Agent协议                                      │
│  新对话 → read 凭据中心.md → 知道结构                      │
│  需要值 → run_command "Get-Content secrets.env" → 获取值   │
│  禁止   → Memory中存储实际密码/token                       │
├─────────────────────────────────────────────────────────┤
│  Layer 3: 凭据中心.md（本文件）                             │
│  git tracked · 所有Agent可读 · 只有键名+描述，无值          │
├─────────────────────────────────────────────────────────┤
│  Layer 2: secrets.env（统一凭据文件）                       │
│  gitignored · 双机各存一份 · sync脚本同步 · 所有实际值      │
├─────────────────────────────────────────────────────────┤
│  Layer 1: 已有config（向后兼容）                            │
│  智能家居/config.json · 阿里云/secrets.toml                │
│  继续运行，但secrets.env是上游源                            │
└─────────────────────────────────────────────────────────┘
```

### secrets.env 与已有配置的关系

| 已有文件 | 角色 | 关系 |
|---------|------|------|
| `阿里云服务器/secrets.toml` | FRP专用配置 | secrets.env的FRP子集，保持同步 |
| `智能家居/网关服务/config.json` | 网关运行时配置 | 从secrets.env初始化，运行时自管理 |
| `双电脑互联/连接卡.md` | Agent操作手册 | 凭据值替换为 `[见secrets.env]` |

## 四、Agent协议（铁律）

### 4.1 读取凭据

```
1. read_file("凭据中心.md")          → 了解有哪些凭据、键名、用途
2. run_command("Get-Content secrets.env")  → 获取实际值
3. 使用后不存入Memory                     → 避免散落
```

### 4.2 铁律五条

| # | 铁律 | 原因 |
|---|------|------|
| **R1** | **Memory禁止存储实际密码/Token值** | Memory跨对话可见，等于公开 |
| **R2** | **git tracked文件禁止明文凭据** | git历史永久可见 |
| **R3** | **新增凭据必须加入secrets.env + 更新本索引** | 防止新的散落 |
| **R4** | **修改凭据只改secrets.env一处** | 单一真相源 |
| **R5** | **Memory中只存键名引用** | 如："`台式机密码见secrets.env DESKTOP_PASSWORD`" |

### 4.3 Memory正确写法示例

```
❌ 错误：台式机密码 ********
✅ 正确：台式机凭据见 secrets.env (DESKTOP_USER/DESKTOP_PASSWORD)

❌ 错误：HA Token: eyJhbGci...（实际值）
✅ 正确：HA认证见 secrets.env (HA_TOKEN)

❌ 错误：微信appsecret: d64dfb4c...（实际值）
✅ 正确：微信凭据见 secrets.env (WECHAT_APPID/WECHAT_APPSECRET/WECHAT_TOKEN)
```

### 4.4 脚本读取模板

**PowerShell**:
```powershell
# 加载secrets.env到环境变量
$envFile = Join-Path $PSScriptRoot "..\secrets.env"
if (Test-Path $envFile) {
    Get-Content $envFile | ForEach-Object {
        if ($_ -match '^\s*([^#=]+?)\s*=\s*(.+)$') {
            [Environment]::SetEnvironmentVariable($Matches[1], $Matches[2], 'Process')
        }
    }
}
$cred = New-Object PSCredential($env:DESKTOP_USER, (ConvertTo-SecureString $env:DESKTOP_PASSWORD -AsPlainText -Force))
```

**Python**:
```python
import os
from pathlib import Path

def load_secrets(env_file=None):
    """从secrets.env加载凭据到环境变量"""
    if env_file is None:
        env_file = Path(__file__).resolve().parent.parent / "secrets.env"
    if env_file.exists():
        for line in open(env_file, encoding='utf-8'):
            line = line.strip()
            if line and not line.startswith('#') and '=' in line:
                k, v = line.split('=', 1)
                os.environ.setdefault(k.strip(), v.strip())

# 使用
load_secrets()
ha_token = os.environ.get("HA_TOKEN", "")
```
