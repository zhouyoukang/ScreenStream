# 共享知识目录

> Developer Cascade 和 Phone Agent 之间的知识交换通道。
> 两个 Windows 账户通过此目录共享信息。

## 文件说明

| 文件 | 写入方 | 读取方 | 内容 |
|------|--------|--------|------|
| api-issues.md | Phone Agent | Developer | API Bug 报告 |
| feature-requests.md | Phone Agent | Developer | 新功能/新 API 需求 |
| device-compat.md | Phone Agent | Developer | 设备兼容性记录 |
| operation-patterns.md | Phone Agent | 双方 | 已验证的操作模式 |

## 格式约定

每条记录包含：日期、设备、APP、描述、重现步骤。
