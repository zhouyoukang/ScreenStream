> 一个链接，帮爸妈解决手机问题。

## 快速开始

### 方式一：链接直连（推荐）

父母手机开启投屏后会显示一个6位数字（如 `123456`），在浏览器打开：

```
https://aiotvr.xyz/cast/?room=123456
```

把数字替换成父母告诉你的数字即可。

### 方式二：扫码连接

父母手机上会显示一个二维码，用手机扫描即可打开。

### 方式三：手动输入

1. 打开浏览器，访问 `https://aiotvr.xyz/cast/`
2. 输入父母手机上显示的6位连接码
3. 点击【连接】

## 操作说明

### 基本操作

| 操作 | 方法 |
|------|------|
| **点击** | 在画面上点击对应位置 |
| **滑动** | 在画面上按住并滑动 |
| **长按** | 在画面上长按不放（约1秒） |
| **返回** | 点击底部【返回】按钮，或按键盘 `Esc` |
| **回主屏** | 点击底部【主屏幕】按钮，或按键盘 `Home` |
| **打字** | 点击底部【键盘】按钮，输入文字后点发送 |
| **音量** | `Ctrl+↑` 增大 / `Ctrl+↓` 减小 |
| **滚动** | 鼠标滚轮上下滚动 |

### 底部导航栏

- **主屏幕** — 回到手机桌面
- **返回** — 相当于手机返回键
- **最近** — 打开最近使用的APP
- **键盘** — 弹出文字输入框
- **更多** — 音量、锁屏、截屏等高级功能

| 状态 | 含义 |
|------|------|
| 🟢 **已连接** | 正常使用中 |
| 🟡 **正在连接** | 正在建立连接，请等待 |

## 首次设置（帮父母配置一次即可）

### 1. 安装APP

将 ScreenStream APK 安装到父母手机上（可通过微信传文件）。

### 2. 开启无障碍服务

这一步让APP能够接收你的远程操作指令：

1. 打开手机 **设置** → **无障碍** → **已安装的服务**
2. 找到 **ScreenStream** → 开启
3. 确认弹窗点 **允许**

> 不同品牌路径略有差异，详见品牌适配表。

### 3. 电池优化白名单

防止系统杀掉后台服务：

1. **设置** → **电池** → **电池优化**（或"省电策略"）
2. 找到 **ScreenStream** → 选择 **不优化**

### 4. 自启动权限（小米/华为/OPPO/vivo）

1. **设置** → **应用管理** → **ScreenStream** → **自启动** → 开启
2. 或 **设置** → **电池** → **后台管理** → 允许后台运行

## 常见问题

**Q: 连接不上怎么办？**
> 1. 确认父母手机已开启投屏（屏幕上有6位数字）
> 2. 确认数字输入正确
> 3. 确认双方网络正常（WiFi或4G）
> 4. 刷新页面重试

**Q: 画面卡顿怎么办？**
> 网络不稳定时可能卡顿，P2P直连模式下体验最佳。
> 如果持续卡顿，建议双方都连WiFi。

**Q: 能听到手机的声音吗？**
> 当前版本主要支持画面+控制，音频传输在后续版本中。

**Q: 安全吗？别人能偷看吗？**
> 安全。连接码每次随机生成，传输全程加密(DTLS-SRTP)，
> 不经过任何第三方服务器。

**Q: 支持哪些浏览器？**
> Chrome、Edge、Firefox、Safari均支持。推荐Chrome/Edge。
