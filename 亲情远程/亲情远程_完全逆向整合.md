## 二、解法·无服务器架构

### 老子·反者道之动

> 不是搭建更多服务器，而是消除服务器。
> 不是对抗NAT，而是利用免费基础设施绕过NAT。
> 不是重新造轮子，而是骑现成的车。

### 新架构: 零自建服务器 · 五级降级

```
手机(ScreenStream)                    子女(浏览器)
    │                                     │
    ├─ L1: WebRTC P2P ────────────────────┤  零服务器, <100ms
    │      (10 STUN + 4 TURN)             │
    ├─ L2: WebRTC via TURN ───────────────┤  免费TURN, <200ms
    │      (metered.ca + openrelay)       │
    ├─ L3: DroidVNC-NG → noVNC ──────────┤  VNC协议, <150ms
    │      (Tailscale免费穿透)             │
    ├─ L4: 向日葵/ToDesk端口转发 ─────────┤  P2P隧道, <100ms
    │      (免费个人版)                    │
    └─ L5: CloudRelay WS中继 ────────────┤  自建中继, ~300ms
           (可选, 非必需)                  │
```

### 信令解决方案 (去服务器化)

| 方案 | 成本 | 维护 | 冷启动 | 适合 |
|------|------|------|--------|------|
| **Cloudflare Workers** | ¥0(100K/天) | 零 | <50ms | ⭐最佳: 全球CDN, WebSocket(Durable Objects) |
| **Render.com** | ¥0(750h/月) | 零 | ~30s | ⭐次佳: 一键部署, auto-sleep |
| **Fly.io** | ¥0(3VM) | 低 | <1s | ⭐好: 永不睡眠, 256MB |
| **GitHub Pages + Firebase** | ¥0 | 零 | 即时 | 好: Viewer静态托管 + Firebase信令 |
| **PeerJS公共服务器** | ¥0 | 零 | 即时 | 好: 但需改Android端协议 |

**推荐**: **Render.com免费部署**(零改动, 同协议) + **Cloudflare Pages托管Viewer**(全球CDN)

### DroidVNC-NG整合方案

DroidVNC-NG是本项目最被低估的资源:
- **Play Store安装** → 绕过Android 13+受限设置(✅解决P3)
- **标准VNC协议** → noVNC浏览器直连(✅无需WebRTC)
- **MediaProjection** → 系统级屏幕捕获(✅全品牌)
- **触控注入** → VNC原生支持(✅无需AccessibilityService)
- **密码保护** → VNC密码认证(✅安全)

**限制**: VNC需要网络可达(需穿透), 无内置P2P
**解决**: Tailscale免费版(100设备, 零配置VPN, MagicDNS)

## 五、免费部署指南

### Step 1: 信令服务器 → Render.com (5分钟)

1. Fork/Push `signaling-server/` 到GitHub
2. 登录 render.com → New Web Service → 选择GitHub仓库
3. 设置:
   - Build: `npm install`
   - Start: `node server.js`
   - Environment: `PORT=10000` (Render分配)
4. 免费tier: 750h/月, 自动部署
5. 获得URL: `wss://your-app.onrender.com/signal/`

### Step 2: Viewer → Cloudflare Pages (3分钟)

1. Push `viewer/` 到GitHub
2. 登录 pages.cloudflare.com → Create project → 选择GitHub仓库
3. Build: 无(纯静态)
4. 获得URL: `https://your-app.pages.dev/`

### Step 3: 更新Viewer信令URL

修改 `viewer/index.html` 中的 `SIGNALING_URL` 指向 Render.com 地址。

### Step 4 (可选): DroidVNC-NG + Tailscale

1. 手机安装 DroidVNC-NG (Play Store)
2. 手机+PC都安装 Tailscale (免费)
3. 子女浏览器打开 `http://[tailscale-ip]:5900` 用noVNC
