## ⚙️ 技术说明

### 采集方法
- 纯ADB操控: `monkey`启动APP + `uiautomator dump`读取UI树 + `input`点击/滑动
- 无需ROOT, 无需ScreenStream API

### 各APP限制原因
| APP | 限制 | 技术原因 |
|-----|------|----------|
| 京东 | 滑动拼图验证码 | 访问"全部订单"触发反自动化机制 |
| 闲鱼 | Flutter渲染 | 列表内容不暴露在accessibility tree中 |
| 饿了么 | UI导航失败 | "我的"页面订单入口文本不匹配 |
| 当当 | 启动广告 | StartupActivity无可点击跳过按钮 |
| 拼多多 | `am start`被拦截 | OPPO自启动限制, 仅monkey可绕过 |

### 解决方案建议
- **京东**: 手动在手机上完成一次滑动验证后, 重新运行脚本即可采集
- **闲鱼**: 需使用截屏+OCR方案, 或通过闲鱼网页版导出
- **饿了么**: 手动打开到订单页后, 运行 `python deep_scroll.py 饿了么` 即可采集
