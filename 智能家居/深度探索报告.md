### 3.4 塑码密钥：execute-text-directive

**这是万物互联的终极能力**——向音箱发送文本指令，音箱在本地WiFi内执行：

```
POST /micloud/rpc
{
  "did": "795748340",       // 小爱音箱Pro
  "siid": 5,                // intelligent-speaker
  "aiid": 5,                // execute-text-directive
  "in": [{"piid":1, "value":"打开灯带"}, {"piid":2, "value":""}]
}
```

**能控制的范围**：
- ✅ WiFi内所有小米/米家设备（含离线设备——音箱通过局域网直接控制）
- ✅ 场景宏（"我要睡觉了"→一键关灯+关风扇）
- ✅ 设备状态查询（"灯带开了吗"）
- ✅ 定时器（"5分钟后关灯"）
- ❌ 非小米设备（eWeLink/Tuya需各自平台API）


### 模式二：存在感知（认知无感）

**来源**：世间最佳实践（basnijholt 100+自动化 + SmartHomeScene mmWave评测）

**我们已有**：小米人在传感器(sensor_occupy.03) — PIR+mmWave双模，静止检测✅

**自动化配方**（HA YAML）：
```yaml
# 人来开灯（30s内无人自动关）
automation:
  - alias: "人来开灯带"
    trigger:
      - platform: state
        entity_id: binary_sensor.xiaomi_sensor_occupy_03_occupancy
        to: "on"
    condition:
      - condition: numeric_state
        entity_id: sensor.xiaomi_sensor_occupy_03_illuminance
        below: 30    # 光照低于30lux才开
    action:
      - service: light.turn_on
        target:
          entity_id: light.philips_strip3
        data:
          brightness_pct: 80

  - alias: "人走关灯带"
    trigger:
      - platform: state
        entity_id: binary_sensor.xiaomi_sensor_occupy_03_occupancy
        to: "off"
        for: "00:05:00"    # 离开5分钟后关
    action:
      - service: light.turn_off
        target:
          entity_id: light.philips_strip3
```

### 15.1 协议层全景（万物通信之基）

| 协议 | 频段 | 范围 | 速度 | 功耗 | 我们用了？ | 取舍 |
|------|------|------|------|------|----------|------|

| **BLE Mesh** | 2.4GHz | 10m | 低 | 极低 | ✅ 温湿度计/人在传感器 | **辅助** |
